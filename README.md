# WebdriverIO/Jenkins

Demonstrate the usage of WebdriverIO in a Jenkins pipeline.

## Getting Started

### Prerequisites

- Node.js (LTS or newer)
- NPM

### Installing

Install dependencies

```bash
npm install
```

### Testing

Run tests

```bash
npm test
```

Run reporting

```bash
npm run test:report
```
